# admin-system-demo(Vue 3 + TypeScript + Vite)
基于Vue3+TypeScript的后台管理系统模板

### Node版本
```
"node": "v18.18.0"
```

#### 从 `GitHub` 上拉取

```bash
git clone https://github.com/duanfanchao/admin-system-demo.git
```

### 安装依赖

```bash
cd admin-system-demo

npm install
```

### 启动

```bash
npm dev
```

### 项目打包

```bash
npm build
```
